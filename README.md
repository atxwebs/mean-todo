# MEAN-TODO
#### TODO list app built with MongoDB + Express + Angular + Node.js

### Clone Repository
```
git clone https://github.com/sethbergman/mean-todo.git && cd mean-todo
```
### Install Dependencies
```
npm install
```
### Start the Server
```
npm start
```
Visit [http://0.0.0.0:5000](http://0.0.0.0:5000) to see the app!
